# useTimeout

# API

```typescript
useTimeout(callback: () => void, ms: number);
```

# 参数

| 参数名   | 说明     | 类型       |
| -------- | -------- | ---------- |
| callback | 回调方法 | () => void |
| ms       | 延时     | number     |
